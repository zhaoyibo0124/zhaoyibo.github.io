2019前端面试题汇总–

一、html部分
1、meta的作用：提供文档字符集、使用语言、作者等基本信息，以及对关键词和网页等级的设定等，可以设置页面缓存和移动端的设置和不使用js刷新页面：http-equiv="refresh" , content="time" 

2、行内元素：`img a input span`   块级元素 `div ul li p table` 

3、css3伪类：`after before  disabled enabled checked`


二、css部分
1、垂直水平居中：
display:flex; just-content:center; align-items:center; position:absolute; top:50%; left:50%; transform:translate(-50%, -50%);
2、清除浮动：使用伪类
content:'';display:block; height:0;clear:both: visibility:hidden

3、visibility和display的区别：visibility会占用空间

4、html5和css3 新增的：语义化标签  `header footer article section nav audio video  transform和animation`开启硬件加速 GPU会耗电

5、超出显示省略号：`over-flow：hidden  text-overflow:ellipsis white-space:nowrap`

6、盒子模型：`conten-box(content+padding+border)和border-box`

三、缓存系列
1、localStorage和sessionStorage区别：本地存储和会话级别;

2、cookie和session的区别: http可设置cookie;

3、浏览器强缓存：cache-control  max-age     协商缓存last-modified if-modified-since   etag if-none-match;

四、页面加载优化
1、压缩js和css 雪碧图 图片懒加载和分段加载 webworker后台加载数据 长轮询改为webscoket

五、js原生
1、栈内存和堆内存：栈内存在当前执行环境结束时会销毁，存放基本数据类型和引用数据类型的对象指针 指向堆内存的地址，堆内存引用他的指针不存在时才会销毁 置为null的原因

2、闭包：指有权访问另一个函数作用域中的变量的函数

3、什么情况会导致内存泄露：闭包，定时器循环  es6的WeakMap和WeakSet可以避免内存泄露  只要外部的引用消失，WeakMap 内部的引用，就会自动被垃圾回收清除

4、改变this指向的方法：bind call apply的区别   参数传参的区别  和立即执行改变this指向的

5、js继承的方法：

   `function Person (name, age) {
       this.name = name,
           this.age = age
   }
   Person.prototype.setAge = function () {
       console.log("111")
   }
   function Student (name, age, price) {
       Person.call(this, name, age)
       this.price = price
       this.setScore = function () { }
   }
   Student.prototype = Object.create(Person.prototype)//核心代码
   Student.prototype.constructor = Student//核心代码`

4、返回此对象的引用地址
九、event-loop事件循环
JavaScript的事件分两种，宏任务(macro-task)和微任务(micro-task)
宏任务：包括整体代码script，setTimeout，setInterval
微任务：Promise.then(非new Promise)，process.nextTick(node中)

　　1）所有同步任务都在主线程上执行，形成一个执行栈
　　2）当主线程中的执行栈为空时，检查事件队列是否为空，如果为空，则继续检查；如不为空，则执行3
　　3）取出任务队列的首部，加入执行栈
　　4）执行任务
　　5）检查执行栈，如果执行栈为空，则跳回第 2 步；如不为空，则继续检查
